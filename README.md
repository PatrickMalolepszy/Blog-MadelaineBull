# [http://madelainebull.com](http://madelainebull.com)

Simple blog set up for Madelaine Bull using full stack Golang and a bootstrap theme provided by [Start Bootstrap](https://startbootstrap.com/template-overviews/clean-blog/).

## Usage

Only uses Go's standard libraries and standard template engine.

To run project:

```
go build -o blogServer
./blogServer
```
